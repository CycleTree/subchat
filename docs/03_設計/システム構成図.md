# システム構成図

## アーキテクチャ

```
┌─────────────────────────────────────────────────────────────┐
│                      SubChat UI                             │
│              (React + TypeScript + MUI)                     │
│                  GitHub Pages                               │
└────────────────────────┬────────────────────────────────────┘
                         │ WebSocket
                         ▼
┌─────────────────────────────────────────────────────────────┐
│                  OpenClaw Gateway                           │
│                     (Fly.io)                                │
├─────────────────────────────────────────────────────────────┤
│  Sessions │ Messages │ Agents │ Auth                        │
└─────────────────────────────────────────────────────────────┘
```

## データフロー

```
User → SubChat UI → WebSocket → Gateway → Session
                                    ↓
                              Agent Response
                                    ↓
                 SubChat UI ← WebSocket ← Gateway
```
