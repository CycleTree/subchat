# SubChat 機能要件

## 概要
OpenClaw サブエージェントの会話を可視化・操作するツール

---

## Must (必須)

### M-1: セッション一覧表示
- OpenClaw Gateway からアクティブセッション一覧を取得
- セッション名、エージェント名、最終更新時刻を表示
- リアルタイム更新

### M-2: メッセージ表示
- 選択したセッションの会話履歴を表示
- ロール別スタイリング (user/assistant/system)
- タイムスタンプ表示

### M-3: メッセージ送信
- 選択したセッションにメッセージを送信
- 送信中/送信完了/エラー状態の表示

### M-4: WebSocket 接続
- OpenClaw Gateway への認証付き WebSocket 接続
- 自動再接続
- 接続状態インジケータ

### M-5: ダークモード
- テーマ切り替えボタン
- localStorage で設定永続化
- システム設定との連携

---

## Should (重要)

### S-1: メッセージコピー
- メッセージをクリップボードにコピー
- コピー成功フィードバック

### S-2: 長文メッセージ折りたたみ
- 一定文字数以上のメッセージを折りたたみ
- 展開/折りたたみ切り替え

### S-3: Agent 階層表示
- 親子関係のツリー表示
- サブエージェント spawn の可視化

### S-4: セッション検索
- セッション名/エージェント名で検索
- フィルタリング

---

## Could (あれば良い)

### C-1: セッション介入
- 実行中セッションへの割り込み
- 追加指示の送信

### C-2: キルスイッチ
- セッションの強制終了
- 確認ダイアログ

### C-3: ログ検索
- メッセージ内容で全文検索
- 日時フィルタ

### C-4: コスト表示
- トークン使用量の表示
- API コスト概算

---

## Won't (対象外)

### W-1: マルチユーザー
- 認証/認可は OpenClaw Gateway に依存
- SubChat 自体のユーザー管理は不要

### W-2: メッセージ編集
- 送信済みメッセージの編集は不要

### W-3: ファイルアップロード
- 画像/ファイル添付は対象外
